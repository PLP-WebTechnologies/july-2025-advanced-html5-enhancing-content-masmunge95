Program Modules

The 3 main modules covered from week to week are as follows:


(i) Web Development V2

Focuses on HTML5, CSS3, and JavaScript, including responsive design and modern frameworks.

(ii) Python Basics

Introduces Python programming, covering syntax, data types, and control structures.

(iii) Database Design & Structures

Covers the fundamentals of database design, including entity-relationship modeling and normalization.
